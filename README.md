## Overview

This repository contains the full client-facing web platform for the **UNMASK Leadership Architecture™** — a diagnostic and engagement system that helps CHROs, COOs, CEOs, and organizational leaders identify and recalibrate the structural design flaws that drive talent collapse, leadership burnout, and attrition.

This is not a wellness tool. This is structural engineering for organizations.

## The Problem We Solve

Burnout is not a personal failure. It is a **design flaw** — the result of misaligned authority, invisible labor, escalation loops, and AI oversight loads that exceed human capacity. UNMASK quantifies these fractures and provides a clear recalibration roadmap.

## The Five Diagnostic Variables

The audit engine scores organizations across five weighted structural variables:

| Variable | Weight | Description |
|---|---|---|
| Authority Gap | 3.0x | Delta between accountability and decision-rights |
| Operational Squeeze | 2.0x | Current load vs. sustainable human capacity |
| Culture of Masking | 1.0x | Pressure to hide structural friction |
| Escalation Density | 2.5x | Frequency of routine tasks requiring senior sign-off |
| AI Oversight Load | 2.5x | Invisible labor created by automation verification |

**Total Weight: 11.0 | Score Range: 1.00 – 5.00**
